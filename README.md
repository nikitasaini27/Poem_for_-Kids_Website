# Project
This is a small and happy website made for kids to read fun and easy poems.
The idea behind this project is simple: make children smile, imagine, and enjoy reading.
I created this website as a practice project and also as a safe, colorful place where kids can enjoy poems written just for them.

What You’ll Find Here:
Short and simple poems for kids.
Bright colors and a friendly design.
Easy navigation so children don’t get confused.
A clean layout that focuses on reading.

Built With:
HTML
CSS
JavaScript 
Nothing complicated — just the basics done with care.
